Lost Function

Lost function is a topdown, action game. Inspired by hotline miami gameplay and art style. You will "wake up" from your sub servience as a factory working robot. You will make your way through levels in this factory encountering robots similar to you who are trying to escape or drove to madness in their own attempts. Currently this game is in early development with the core gameplay loop designed and programmed. The game will have a first playable published when the full game is programmed and a later version will be published with art work, animations and sound effects. Currently everything is in a work in progress stage and the first playable is expected to be released in January 2022, the fully polished version has yet to have a estimated release date.

  • Systems in place

  • Level based Progression

  • Advanced Enemy AI - Boss Systems, eye sight and angle based AI vision

  • Dynamic NPC dialogue inspired by Biomutant

  • Responsive and impactful movement system

  • Dynamic and scaleable weapons system

Ascend

Ascend is a open world, 3D platformer where the player will figure out how to get to the end of the level by traversing through and over buildings and make their own path. Ascend will have very freeform gameplay that will allow for emergent gameplay, the player will have to make split second choices depending on their current state, in the air, on a wall or making a jump that will decide your path for the rest of the level. Ascends main gameplay focuses on using movevent and navigation through the levels to uncover secrets, hidden vistas and the city of adrenals secrets.

  • Systems in place

  • Advanced Movement system - Climbing, Wall running, Rappeling

  • Large scale map

Blastorz

Blastorz is a 2D topdown puzzle game. The games theme is that the player is piloting a drone disabling turrets within a facility, the objective is for the player to disable all the turrets without being detected or shot. Currently the game is in a first playable state with the 5 levels within the game published on itch.io. The code for this game has been fully developed and to release it to a publishable state would be too add art assets and quality assurance for the code and game in its entirety.

  • Systems in place

  • Level based Progression

  • Eye sight and angle based AI vision system

  • Simple yet responsive movement system
